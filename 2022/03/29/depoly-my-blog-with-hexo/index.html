配置github

github –> settings –> SSH and GPG keys –> New SSH Key

查看刚才使用ssh生成的公钥，把内容复制到Key中

使用hexo上传时，需要输入github的账户和密码，但github目前无法使用密码登陆了，需要使用Personal access tokens, 获取方法:

github –> settings –> Developer settings –> Personal access tokens –> Generate new token

需要输入密码时输入token就可以
